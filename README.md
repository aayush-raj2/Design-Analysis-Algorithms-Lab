# Design and Analysis of Algorithms – Lab Experiments

This repository contains the **lab experiments for Design and Analysis of Algorithms (DAA)** implemented as part of the BTech CSE curriculum.

The experiments are written with **clear logic, structured code, and time complexity analysis**, making them useful for **lab exams, viva, and revision**.

---

## 📌 Objectives
- Understand core algorithm design techniques
- Implement algorithms efficiently
- Analyze **time and space complexity**
- Prepare for **DAA lab exams and viva**

---

## 📂 Experiments Covered
- Sorting Algorithms
- Divide and Conquer
- Greedy Algorithms
- Dynamic Programming
- Backtracking
- Graph Algorithms

*(Experiments are organized experiment-wise as per lab syllabus)*

---

## 🧪 Experiment Structure
Each experiment contains:
- **Aim**
- **Algorithm**
- **Program Code**
- **Time Complexity**
- **Sample Output**

- Platform: Linux / Windows
