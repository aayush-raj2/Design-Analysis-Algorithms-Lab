### 1ï¸âƒ£ Huffman Coding
Huffman Coding is a greedy algorithm used for data compression.  
It builds an optimal prefix tree based on character frequencies.

Time Complexity: O(n log n)  
Space Complexity: O(n)

Key Concept:
- Always combine two minimum frequency nodes
- Use Min-Heap / Priority Queue

Applications:
- File compression
- ZIP, JPEG encoding concepts

The Fractional Knapsack problem maximizes profit by selecting items based on highest value/weight ratio.

- Sort items by value/weight ratio
- Pick maximum ratio first
- Fractions allowed
