# Travelling Salesman Problem (TSP) – C & C++

## 📌 Overview
This repository contains implementations of the **Travelling Salesman Problem (TSP)** using a brute-force approach.

TSP is a classic optimization problem where a salesman must visit all cities exactly once and return to the starting city with minimum cost.

---

## 🧠 Approach Used

Brute Force (Permutation-Based)

Algorithm:
1. Generate all possible permutations of cities
2. Calculate total travel cost for each path
3. Choose the minimum cost path

---

## ⏱ Complexity

Time Complexity: O(n!)  
Space Complexity: O(n)

---

## 🚀 Features

- Solves TSP for small input sizes
- Demonstrates exponential complexity
- Simple and easy-to-understand implementation
